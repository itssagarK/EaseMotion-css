# Grayscale Filter Utility Showcase

Demonstrates the implementation of the CSS grayscale filter utilities under issue #15201.

## Grayscale Utilities Included

- `.ease-filter-grayscale`: Applies 100% grayscale filter to the target element.
- `.ease-filter-grayscale-0`: Removes any active grayscale filter (setting it to 0).

## Showcase Layout

Open `demo.html` in a web browser. Hovering over the grayscale portrait card dynamically shifts it to full color using standard transitions and `.ease-filter-grayscale-0`.

Closes #15201
