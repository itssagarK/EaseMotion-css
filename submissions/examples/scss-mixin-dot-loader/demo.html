2. SCSS Implementation Code

Here is how the mixin is defined in EaseMotion and used within stylesheet files:

Mixin Definition (scss/_mixins.scss) 
@mixin dot-loader(
  $dot-size: 10px,
  $dot-color: var(--ease-color-primary, #6366f1),
  $dot-spacing: 6px,
  $dot-count: 3,
  $animation-duration: 1.2s,
  $animation-delay-step: 0.15s
) {
  display: inline-flex;
  align-items: center;
  gap: $dot-spacing;

  .ease-dot-loader-item {
    width: $dot-size;
    height: $dot-size;
    background-color: $dot-color;
    border-radius: 50%;
    animation: ease-kf-dot-loader-anim $animation-duration infinite ease-in-out;

    @for $i from 1 through $dot-count {
      &:nth-child(#{$i}) {
        animation-delay: ($i - 1) * $animation-delay-step;
      }
    }
  }
}
Invoking with Custom Arguments (style.scss) 
@use '../../../scss/mixins';

.ease-dot-loader-five {
  // Invokes a 5-dot emerald loading wave
  @include mixins.dot-loader(
    $dot-size: 8px,
    $dot-spacing: 5px,
    $dot-color: #10b981,
    $dot-count: 5,
    $animation-duration: 1.5s,
    $animation-delay-step: 0.12s
  );
}
