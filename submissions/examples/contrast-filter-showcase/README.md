# Contrast Filter Utility Showcase

Demonstrates the implementation of the CSS contrast filter utilities under issue #15204.

## Contrast Utilities Included

- `.ease-filter-contrast`: Sets contrast to standard 100%.
- `.ease-filter-contrast-50`: Lowers contrast to 50%.
- `.ease-filter-contrast-150`: Raises contrast to 150%.

## Showcase Layout

Open `demo.html` in a web browser. Compare cards that apply standard, low-contrast, and high-contrast styling layers dynamically.

Closes #15204
