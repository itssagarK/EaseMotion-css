# Brightness Filter Utility Showcase

Demonstrates the implementation of the CSS brightness filter utilities under issue #15205.

## Brightness Utilities Included

- `.ease-filter-brightness`: Sets brightness to standard 100%.
- `.ease-filter-brightness-50`: Lowers brightness to 50%.
- `.ease-filter-brightness-150`: Raises brightness to 150%.

## Showcase Layout

Open `demo.html` in a web browser. Compare cards that apply standard, low-brightness, and high-brightness styling layers dynamically.

Closes #15205
